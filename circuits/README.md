# Circuitos

Este diretório contém os arquivos de circuitos e esquemáticos desenvolvidos na atividade.

## Formatos Suportados

- Arquivos de esquemáticos (.sch)
- Arquivos de layout (.kicad_pcb, .brd)
- Diagramas (.svg, .png)
- Netlists

## Ferramentas Recomendadas

- KiCad
- LTspice
- Eagle
- Altium Designer

## Organização

Organize os circuitos em subdiretórios por projeto ou componente quando necessário.
