# Relatórios

Este diretório contém os relatórios e análises da atividade.

## Estrutura de Relatório

Um relatório completo deve incluir:

1. **Introdução**
   - Objetivos
   - Escopo do trabalho

2. **Desenvolvimento**
   - Metodologia aplicada
   - Descrição dos circuitos
   - Análise teórica

3. **Resultados**
   - Dados de simulação
   - Gráficos e tabelas
   - Análise dos resultados

4. **Conclusões**
   - Discussão dos resultados
   - Aprendizados
   - Próximos passos

5. **Referências**
   - Bibliografia
   - Datasheets utilizados

## Formato

- Preferência por Markdown (.md) ou LaTeX (.tex)
- PDFs são aceitos para versões finais
- Inclua todas as figuras e referências
