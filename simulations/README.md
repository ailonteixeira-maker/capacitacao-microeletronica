# Simulações

Este diretório contém arquivos de simulação e análise de circuitos.

## Tipos de Simulação

- Simulações SPICE (.cir, .asc)
- Análises transientes
- Análises AC/DC
- Análises paramétricas
- Resultados de simulação (.raw, .txt)

## Ferramentas

- LTspice
- NGSPICE
- HSPICE
- Cadence Spectre

## Estrutura

```
simulations/
├── spice/          # Arquivos SPICE
├── results/        # Resultados e gráficos
└── scripts/        # Scripts de automação
```

## Como Executar

Consulte os comentários em cada arquivo de simulação para instruções específicas de execução.
